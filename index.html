About the Project

What is the Web Arm Controller?

The Web Arm Controller is an IoT system that enables remote operation of a robotic arm directly from any modern web browser — no dedicated software required.

By combining ESP32 firmware, WebSocket communication, and a responsive web dashboard, the system translates user commands into precise servo motor movements with sub-20ms latency over a local Wi-Fi network.

Project Goals

Core Objectives

Six key goals that define the project's scope, direction, and success criteria.

🌐

Remote Web Interface

Develop a responsive control dashboard accessible from any device — no installation, just a URL.

⚡

Real-Time Communication

Implement low-latency WebSocket channels for immediate bi-directional command and status transfer.

🎯

Precise Motion Control

Achieve smooth, accurate servo positioning with configurable speed profiles and IK-based coordinate input.

📊

Live Feedback & Telemetry

Stream real-time joint angles, grip force, camera feed, and system health to the web dashboard.

🛡️

Safety & Limit Protection

Enforce hardware and software joint limits with a one-click emergency stop available in the web UI.

🔧

Scalable Architecture

Design a modular system that can extend to support multiple arms, additional sensors, or automation scripts.

Hardware Components

Physical Build

Every physical component that makes up the Web Arm Controller system.

🦾

6-DOF Robotic Arm Frame

MECHANICAL
▼
Aluminium alloy arm with 6 precision joints and cable channels. Engineered for repeatability and smooth motion across all axes.
⚙️

MG996R Servo Motors × 4 & MG90S Servo Motors x 3

ACTUATOR
▼
High-torque metal gear servos, 180° range, 10 kg·cm per joint. Provides reliable actuation for all arm movements with fast response times.
🟦

ESP32 Microcontroller

MCU
▼
240 MHz dual-core MCU with built-in Wi-Fi — runs the WebSocket server. Handles real-time servo control and sensor communication seamlessly.
🔌

PCA9685 PWM Driver

DRIVER IC
▼
16-channel 12-bit I²C PWM controller for precise servo signal output. Enables synchronized control of all 6 motors independently.
📷

L298 Motor Driver

SENSOR
▼
The L298 motor driver is an electronic module used in Arduino projects to control the speed and direction of DC motors and stepper motors. It allows the Arduino to drive motors safely by providing the required current and voltage.
⚡

12V / 5A Power Supply

POWER
▼
Regulated DC switching supply for stable servo and logic power. Provides sufficient current headroom for all servos operating simultaneously.
🔋

LM2596 Buck Converter

POWER
▼
Steps 12V down to 5V for ESP32 and logic-level peripherals. Ensures stable operation of digital components without noise interference.
📡

DC Motor

POWER
▼
A DC motor is an electrical motor that converts direct current (DC) electrical energy into mechanical rotational motion. It is commonly used in Arduino and robotics projects to create movement in wheels, robotic arms, and other mechanical parts.
🤏

Wheels tire 4 x

PHYSICAL
▼
The 4 wheels are used to provide movement, balance, and stability for the robot platform. They help the robot move smoothly in different directions during operation.
